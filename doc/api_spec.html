API Specification: 회원 & 인증

기업 고객 관리 시스템(CMS) 백엔드 API 규격 테이블 문서

회원 및 인증 관련 API (Member)

HTTP API 경로 권한 등급 기능 설명 및 예제 (Request / Response)
POST /backend/member/register.php 관리자 전용
신규 사용자 계정을 시스템에 등록합니다.
예제 보기
Request Body (JSON)
{ "login_id": "user01", "password": "pass1234", "name": "홍길동" }
Response (JSON)
{ "success": true, "message": "회원가입이 완료되었습니다.", "data": { "id": 5 } }
POST /backend/member/login.php 전체 허용
아이디 및 패스워드로 로그인 처리 후 세션을 생성합니다.
예제 보기
Request Body (JSON)
{ "login_id": "admin", "password": "admin1234" }
Response (JSON)
{ "success": true, "message": "로그인 되었습니다.", "data": { "id": 1, "name": "관리자", "role": "admin" } }
POST /backend/member/logout.php 로그인 회원
현재 로그인 세션을 파괴하고 로그아웃 처리합니다.
예제 보기
Response (JSON)
{ "success": true, "message": "로그아웃 되었습니다." }
GET /backend/member/me.php 로그인 회원
현재 세션의 로그인 사용자 정보(ID, 역할, 이름)를 반환합니다.
예제 보기
Response (JSON)
{ "success": true, "data": { "id": 1, "name": "관리자", "role": "admin" } }
PUT /backend/member/update.php 로그인 회원
내 프로필 정보(이름, 새 비밀번호)를 수정합니다.
예제 보기
Request Body (JSON)
{ "name": "홍길동", "password": "newPassword123" }
Response (JSON)
{ "success": true, "message": "내 정보가 수정되었습니다." }
GET /backend/member/list.php 관리자 전용
가입되어 있는 전체 회원 목록을 조회합니다.
예제 보기
Response (JSON)
{ "success": true, "data": [ { "id": 1, "login_id": "admin", "name": "관리자", "role": "admin", "created_at": "2026-07-10 18:01:31" }, { "id": 2, "login_id": "user01", "name": "사원A", "role": "user", "created_at": "2026-07-11 01:23:45" } ] }
PUT /backend/member/set_role.php 관리자 전용
특정 회원의 시스템 권한 등급을 강제로 조정합니다.
예제 보기
Request Body (JSON)
{ "member_id": 2, "role": "admin" }
Response (JSON)
{ "success": true, "message": "역할 변경이 완료되었습니다." }

고객사 정보 API (Customer)

HTTP API 경로 권한 등급 기능 설명 및 예제 (Request / Response)
GET /backend/customer/list.php 로그인 회원
고객 목록을 반환합니다. 일반 계정은 숨김 상태인 고객이 제외됩니다. (관리자는 전체 및 상태별 통계 포함)
예제 보기
Response (JSON)
{ "success": true, "data": [ { "id": 1, "company_name": "(주)샘플기업", "company_addr": "서울시 강남구 테헤란로 123", "contact_name": "홍길동", "contact_phone": "02-1234-5678", "contact_email": "hong@sample.com", "is_hidden": 0, "created_by": 1, "created_by_name": "관리자", "created_at": "2026-07-10 18:01:31", "updated_at": "2026-07-10 18:01:31" } ] }
GET /backend/customer/get.php 로그인 회원
선택한 특정 고객의 상세 프로필 정보를 조회합니다.
예제 보기
Query Parameters
?id=1
Response (JSON)
{ "success": true, "data": { "id": 1, "company_name": "(주)샘플기업", "company_addr": "서울시 강남구 테헤란로 123", "contact_name": "홍길동", "contact_phone": "02-1234-5678", "contact_email": "hong@sample.com", "is_hidden": 0, "created_by": 1, "created_by_name": "관리자", "created_at": "2026-07-10 18:01:31", "updated_at": "2026-07-10 18:01:31" } }
POST /backend/customer/create.php 로그인 회원
신규 기업 고객 정보를 등록합니다.
예제 보기
Request Body (JSON)
{ "company_name": "주식회사 테크앤로", "company_addr": "서울시 마포구 독막로 45", "contact_name": "임꺽정", "contact_phone": "010-9876-5432", "contact_email": "lim@techlaw.com" }
Response (JSON)
{ "success": true, "message": "고객 정보가 등록되었습니다.", "data": { "id": 4 } }
PUT /backend/customer/update.php 로그인 회원
기존 등록된 특정 고객사 기본 정보를 수정합니다.
예제 보기
Request Body (JSON)
{ "id": 4, "company_name": "주식회사 테크앤로", "company_addr": "서울시 마포구 신촌로 100", "contact_name": "임꺽정", "contact_phone": "010-9876-1234", "contact_email": "lim_new@techlaw.com" }
Response (JSON)
{ "success": true, "message": "고객 정보가 수정되었습니다." }
PUT /backend/customer/hide.php 관리자 전용
특정 고객을 일반 사원 목록에서 안 보이도록 숨기거나 활성화합니다.
예제 보기
Request Body (JSON)
{ "id": 1, "is_hidden": true }
Response (JSON)
{ "success": true, "message": "고객 정보 숨김 설정이 변경되었습니다." }

하드웨어 모델 API (Hardware)

HTTP API 경로 권한 등급 기능 설명 및 예제 (Request / Response)
GET /backend/hardware/list.php 로그인 회원
전체 하드웨어 모델 목록을 반환합니다. 일반 직원은 활성(`is_active=1`) 상태인 모델만 선택 가능합니다.
예제 보기
Response (JSON)
{ "success": true, "data": [ { "id": 1, "model_name": "SuperServer 1U-A100", "cpu_count": 2, "disk_tb": "10.00", "nic_count": 2, "note": "고성능 1U 서버", "is_active": 1, "created_by": 1, "created_by_name": "관리자", "created_at": "2026-07-11 02:00:00", "updated_at": "2026-07-11 02:00:00" } ] }
GET /backend/hardware/get.php 로그인 회원
특정 하드웨어의 상세 명세(CPU 수, DISK, NIC, 비고 등)를 상세 조회합니다.
예제 보기
Query Parameters
?id=1
Response (JSON)
{ "success": true, "data": { "id": 1, "model_name": "SuperServer 1U-A100", "cpu_count": 2, "disk_tb": "10.00", "nic_count": 2, "note": "고성능 1U 서버", "is_active": 1, "created_by": 1, "created_by_name": "관리자", "created_at": "2026-07-11 02:00:00", "updated_at": "2026-07-11 02:00:00" } }
POST /backend/hardware/create.php 관리자 전용
신규 판매 가능 하드웨어 장비 스펙 정보를 카탈로그에 등록합니다.
예제 보기
Request Body (JSON)
{ "model_name": "PowerEdge R760", "cpu_count": 4, "disk_tb": 32.5, "nic_count": 4, "note": "Dell 2U 랙형 고성능 데이터베이스 서버" }
Response (JSON)
{ "success": true, "message": "하드웨어가 등록되었습니다.", "data": { "id": 4 } }
PUT /backend/hardware/update.php 관리자 전용
장비 스펙 사양이나 비고 항목을 갱신합니다.
예제 보기
Request Body (JSON)
{ "id": 4, "model_name": "PowerEdge R760 Enterprise", "cpu_count": 8, "disk_tb": 64.0, "nic_count": 4, "note": "업그레이드 사양 반영" }
Response (JSON)
{ "success": true, "message": "하드웨어 정보가 수정되었습니다." }
PUT /backend/hardware/toggle.php 관리자 전용
하드웨어 장비의 판매 활성/비활성 상태(`is_active` 0/1)를 스위치 토글 전환합니다.
예제 보기
Request Body (JSON)
{ "id": 1 }
Response (JSON)
{ "success": true, "message": "비활성화되었습니다.", "data": { "is_active": false } }

설치 제품 API (Product)

HTTP API 경로 권한 등급 기능 설명 및 예제 (Request / Response)
GET /backend/product/list.php 로그인 회원
해당 기업 고객에게 판매/설치된 제품 목록을 조회합니다.
예제 보기
Query Parameters
?customer_id=1
Response (JSON)
{ "success": true, "data": [ { "id": 1, "customer_id": 1, "hardware_id": 1, "model_name": "CMS Enterprise v2.0", "version": "v2.0.4", "os_type": "Rocky Linux 9.2", "installed_at": "2024-01-15", "created_by": 1, "created_at": "2026-07-11 02:10:00", "updated_at": "2026-07-11 02:10:00", "hw_model": "SuperServer 1U-A100", "created_by_name": "관리자" } ] }
GET /backend/product/get.php 로그인 회원
제품 및 연동된 하드웨어 세부 사양 정보를 단건 조회합니다.
예제 보기
Query Parameters
?id=1
Response (JSON)
{ "success": true, "data": { "id": 1, "customer_id": 1, "hardware_id": 1, "model_name": "CMS Enterprise v2.0", "version": "v2.0.4", "os_type": "Rocky Linux 9.2", "installed_at": "2024-01-15", "created_by": 1, "created_at": "2026-07-11 02:10:00", "updated_at": "2026-07-11 02:10:00", "hw_model": "SuperServer 1U-A100", "hw_cpu": 2, "hw_disk": "10.00", "hw_nic": 2, "created_by_name": "관리자" } }
POST /backend/product/create.php 관리자 전용
고객 프로필에 설치된 신규 제품 정보를 등록합니다. (달력 UI 지원)
예제 보기
Request Body (JSON)
{ "customer_id": 1, "model_name": "CMS Core Web v3.0", "version": "v3.0.0", "os_type": "Ubuntu 22.04 LTS", "hardware_id": 2, "installed_at": "2026-07-11" }
Response (JSON)
{ "success": true, "message": "제품이 등록되었습니다.", "data": { "id": 3 } }
PUT /backend/product/update.php 관리자 전용
고객에게 기설치된 제품 및 OS, 설치일 정보를 갱신합니다.
예제 보기
Request Body (JSON)
{ "id": 3, "hardware_id": 3, "model_name": "CMS Core Web v3.1", "version": "v3.1.0", "os_type": "Ubuntu 24.04 LTS", "installed_at": "2026-07-11" }
Response (JSON)
{ "success": true, "message": "제품 정보가 수정되었습니다." }
DELETE /backend/product/delete.php 관리자 전용
등록된 설치 제품 정보를 삭제합니다.
예제 보기
Request Body (JSON)
{ "id": 3 }
Response (JSON)
{ "success": true, "message": "제품 정보가 삭제되었습니다." }
