API Specification: 회원 & 인증

기업 고객 관리 시스템(CMS) 백엔드 API 규격 테이블 문서

회원 및 인증 관련 API (Member)

HTTP API 경로 권한 등급 기능 설명 및 예제 (Request / Response)
POST /backend/v1/member/register.php 관리자 전용
신규 사용자 계정을 시스템에 등록합니다.
예제 보기
Request Body (JSON)
{ "login_id": "engineer1", "password": "password123", "name": "강동우", "role": "engineer" }
POST /backend/v1/member/login.php 전체 공개
아이디와 비밀번호로 로그인 인증을 진행하고 세션을 생성합니다.
GET /backend/v1/member/me.php 일반 사용자
현재 로그인된 본인 계정의 요약 프로필 세션 정보를 획득합니다.
POST /backend/v1/member/logout.php 일반 사용자
현재 활성화된 계정 세션을 파괴하고 로그아웃합니다.

고객사 관리 관련 API (Customer)

HTTP API 경로 권한 등급 기능 설명 및 예제 (Request / Response)
GET /backend/v1/customer/list.php 일반 사용자
고객사 리스트를 조회합니다. 관리자는 숨겨진 고객까지 수집하며, 일반 사용자는 활성 고객만 받습니다.
GET /backend/v1/customer/get.php 일반 사용자
특정 고객의 상세 정보를 획득합니다.
POST /backend/v1/customer/create.php 일반 사용자
신규 고객 정보를 생성합니다. (기타 사항 description 입력 연동 적용)
PUT /backend/v1/customer/update.php 일반 사용자
지정된 ID의 고객 정보를 갱신합니다. (기타 사항 description 컬럼 갱신 가능)
PUT /backend/v1/customer/hide.php 관리자 전용
지정한 고객사를 목록에서 숨김 처리(is_hidden = 1) 합니다.

설치 제품 관리 관련 API (Product)

HTTP API 경로 권한 등급 기능 설명 및 예제 (Request / Response)
GET /backend/v1/product/list.php 일반 사용자
특정 고객사(customer_id)에 귀속된 설치 제품 목록을 조회합니다.
GET /backend/v1/product/get.php 일반 사용자
지정한 제품의 상세 스펙 정보를 조회하며, 제품 변경/업그레이드(version, OS) 이력 목록을 history 노드로 함께 병합 반환합니다.
POST /backend/v1/product/create.php 관리자 전용
고객사에 신규 소프트웨어 제품 설치 정보를 할당합니다. (name(제품 이름) 컬럼 반영, hardware_id 제거됨)
PUT /backend/v1/product/update.php 관리자 전용
제품 명칭, 설치일자, 비고 및 설명 데이터를 갱신합니다. (스펙 업그레이드 전용 기능 분리 적용)
PUT /backend/v1/product/upgrade.php 관리자 전용
소프트웨어의 버전(MAGUX_VER) 및 OS 스펙을 전용 업그레이드 조치하며, 변경 이전 값을 대조하여 `product_history` 테이블에 참고사항(notes)과 함께 이력을 적재합니다.
DELETE /backend/v1/product/delete.php 관리자 전용
고객사의 설치 제품 정보를 영구 제거합니다.

정기 점검 관리 관련 API (Inspection)

HTTP API 경로 권한 등급 기능 설명 및 예제 (Request / Response)
GET /backend/v1/inspection/list.php 일반 사용자
정기 점검 목록을 조건 필터링(customer_id, status, member_id)하여 조회합니다.
GET /backend/v1/inspection/get.php 일반 사용자
특정 점검 계획의 상세 정보 및 배정 담당 엔지니어 리스트를 조회합니다.
POST /backend/v1/inspection/create.php 일반 사용자
신규 정기 점검 계획을 등록하고 담당 엔지니어를 다중 배정합니다.
PUT /backend/v1/inspection/update.php 일반 사용자
등록된 점검 계획을 수정하거나 점검 완료 처리 및 결과 보고서/이슈사항을 등록합니다.
DELETE /backend/v1/inspection/delete.php 관리자 전용
특정 점검 계획 및 이력 매핑 정보를 완전히 삭제합니다.

대항목 카테고리 관리 API (Category)

HTTP API 경로 권한 등급 기능 설명 및 예제 (Request / Response)
GET /backend/v1/mng_category/list.php 일반 사용자
등록된 대항목 카테고리 코드(e.g., HW, OS, MAGUX_VER) 전체 목록을 로드합니다.
POST /backend/v1/mng_category/create.php 관리자 전용
신규 대항목 카테고리(code, name, description)를 저장합니다.
PUT /backend/v1/mng_category/update.php 관리자 전용
기존 대항목 카테고리의 정보(이름, 코드, 설명)를 수정합니다.
DELETE /backend/v1/mng_category/delete.php 관리자 전용
지정한 대항목 카테고리를 시스템에서 제거합니다. (하위 세부 항목들이 연쇄 삭제됩니다)

세부 항목 관리 API (Item)

HTTP API 경로 권한 등급 기능 설명 및 예제 (Request / Response)
GET /backend/v1/mng_item/list.php 일반 사용자
특정 대항목에 속한 세부 옵션 리스트를 수집합니다. category_id 또는 category_code(예: MAGUX_VER)로 조회할 수 있습니다.
POST /backend/v1/mng_item/create.php 관리자 전용
특정 대항목 아래에 신규 세부 코드 옵션(code, name, is_use)을 등록합니다.
PUT /backend/v1/mng_item/update.php 관리자 전용
지정한 세부 코드 항목 정보(이름, 코드, 사용 여부)를 수정합니다.
DELETE /backend/v1/mng_item/delete.php 관리자 전용
선택한 세부 옵션 항목을 데이터베이스에서 완전히 삭제합니다.
