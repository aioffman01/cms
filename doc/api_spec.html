CMS API Specification Reference

기업 고객 관리 시스템(CMS) 백엔드 API 규격 테이블 문서

1. 회원 및 인증 관련 API (Member)

HTTP API 경로 권한 등급 요청 파라미터 (JSON Body / Query) 기능 설명
POST /backend/member/register.php 관리자 전용 login_id: string (아이디)
password: string (비밀번호)
name: string (이름)		 신규 사용자 계정을 시스템에 등록합니다.
POST /backend/member/login.php 전체 허용 login_id: string
password: string		 아이디 및 패스워드로 로그인 처리 후 세션을 생성합니다.
POST /backend/member/logout.php 로그인 회원 - 현재 로그인 세션을 파괴하고 로그아웃 처리합니다.
GET /backend/member/me.php 로그인 회원 - 현재 세션의 로그인 사용자 정보(ID, 역할, 이름)를 반환합니다.
PUT /backend/member/update.php 로그인 회원 name: string (필수)
password: string (선택)		 내 프로필 정보(이름, 새 비밀번호)를 수정합니다.
GET /backend/member/list.php 관리자 전용 - 가입되어 있는 전체 회원 목록을 조회합니다.
PUT /backend/member/set_role.php 관리자 전용 member_id: int (회원 ID)
role: string ("admin"|"user")		 특정 회원의 시스템 권한 등급을 강제로 조정합니다.

2. 고객 정보 및 설치 제품 API (Customer & Product)

HTTP API 경로 권한 등급 요청 파라미터 (JSON Body / Query) 기능 설명
GET /backend/customer/list.php 로그인 회원 - 고객 목록을 반환합니다. 일반 계정은 숨김 상태인 고객이 제외됩니다. (관리자는 전체 및 상태별 통계 포함)
GET /backend/customer/get.php 로그인 회원 id: int (고객 고유 ID, Query) 선택한 특정 고객의 상세 프로필 정보를 조회합니다.
POST /backend/customer/create.php 로그인 회원 company_name: string (필수)
company_addr: string
contact_name: string
contact_phone: string
contact_email: string		 신규 기업 고객 정보를 등록합니다.
PUT /backend/customer/update.php 로그인 회원 id: int (고객 ID)
company_name: string (필수)
company_addr, contact_name 등		 기존 등록된 특정 고객사 기본 정보를 수정합니다.
PUT /backend/customer/hide.php 관리자 전용 id: int (고객 ID)
is_hidden: boolean (숨김 여부)		 특정 고객을 일반 사원 목록에서 안 보이도록 숨기거나 활성화합니다.
GET /backend/product/list.php 로그인 회원 customer_id: int (고객 ID, Query) 해당 기업 고객에게 판매/설치된 제품 목록을 조회합니다.
GET /backend/product/get.php 로그인 회원 id: int (제품 ID, Query) 제품 및 연동된 하드웨어 세부 사양 정보를 단건 조회합니다.
POST /backend/product/create.php 관리자 전용 customer_id: int
model_name: string (필수)
license: string
os_type: string (설치 OS)
hardware_id: int (선택)
installed_at: string (설치일)		 고객 프로필에 설치된 신규 제품 정보를 등록합니다. (달력 UI 지원)
PUT /backend/product/update.php 관리자 전용 id: int (제품 ID)
model_name: string (필수)
license, os_type, hardware_id, installed_at		 고객에게 기설치된 제품 및 OS, 설치일 정보를 갱신합니다.
DELETE /backend/product/delete.php 관리자 전용 id: int (제품 ID) 등록된 설치 제품 정보를 삭제합니다.

3. 하드웨어 모델 API (Hardware)

HTTP API 경로 권한 등급 요청 파라미터 (JSON Body / Query) 기능 설명
GET /backend/hardware/list.php 로그인 회원 - 전체 하드웨어 모델 목록을 반환합니다. 일반 직원은 활성(`is_active=1`) 상태인 모델만 선택 가능합니다.
GET /backend/hardware/get.php 로그인 회원 id: int (하드웨어 ID, Query) 특정 하드웨어의 상세 명세(CPU 수, DISK, NIC, 비고 등)를 상세 조회합니다.
POST /backend/hardware/create.php 관리자 전용 model_name: string (필수)
cpu_count: int
disk_tb: float
nic_count: int
note: string		 신규 판매 가능 하드웨어 장비 스펙 정보를 카탈로그에 등록합니다.
PUT /backend/hardware/update.php 관리자 전용 id: int (하드웨어 ID)
model_name: string (필수)
cpu_count, disk_tb, nic_count, note 등		 장비 스펙 사양이나 비고 항목을 갱신합니다.
PUT /backend/hardware/toggle.php 관리자 전용 id: int (하드웨어 ID) 하드웨어 장비의 판매 활성/비활성 상태(`is_active` 0/1)를 스위치 토글 전환합니다.
