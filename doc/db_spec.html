Database Specification: member

선택한 테이블의 스키마 명세 및 설명

회원 테이블 (member)

Table Schema: member
필드명 데이터 타입 Null Key 기본값 설명 / 비고
id INT UNSIGNED NO PRI (Auto Inc) NULL 고유 식별 회원 번호 (PK)
login_id VARCHAR(50) NO UNI None 로그인 아이디 (중복 불가)
password VARCHAR(255) NO - None 해싱 암호화된 비밀번호 (bcrypt)
name VARCHAR(100) NO - None 사용자 이름
role VARCHAR(20) NO - 'engineer' 회원 등급 구분 ('admin' = 관리자, 'engineer' = 일반엔지니어)
created_at DATETIME NO - CURRENT_TIMESTAMP 가입 일자 및 시각
updated_at DATETIME NO - CURRENT_TIMESTAMP 회원 정보 수정 시각

고객사 테이블 (customer)

Table Schema: customer
필드명 데이터 타입 Null Key 기본값 설명 / 비고
id INT UNSIGNED NO PRI (Auto Inc) NULL 고객사 고유 ID (PK)
company_name VARCHAR(200) NO MUL (Index) None 고객사 회사 이름
company_addr VARCHAR(500) NO - '' 회사 소재지 주소
contact_name VARCHAR(100) NO - '' 고객사 담당 대표자 이름
contact_phone VARCHAR(20) NO - '' 대표 연락처 전화번호
contact_email VARCHAR(200) NO - '' 대표 이메일 주소
description TEXT YES - NULL 고객 등록 관련 기타 특이사항 메모
is_hidden TINYINT(1) NO MUL (Index) 0 숨김 처리 플래그 (0 = 정상 노출, 1 = 목록 숨김)
created_by INT UNSIGNED NO MUL (FK) None 최초 등록 임직원 ID (REFERENCES member(id))
created_at DATETIME NO MUL (Index) CURRENT_TIMESTAMP 정보 등록 시각
updated_at DATETIME NO - CURRENT_TIMESTAMP 정보 수정 시각

설치 제품 테이블 (product)

Table Schema: product
필드명 데이터 타입 Null Key 기본값 설명 / 비고
id INT UNSIGNED NO PRI (Auto Inc) NULL 판매 설치 제품 일련번호 (PK)
customer_id INT UNSIGNED NO MUL (FK) None 고객사 ID (REFERENCES customer(id) ON DELETE CASCADE)
name VARCHAR(255) NO - None 설치된 소프트웨어 개별 제품 명칭
model_name VARCHAR(200) NO - None 설치된 소프트웨어 제품 모델 이름 (Dropdown 로드)
version VARCHAR(500) NO - '' 소프트웨어 마구스 버전 (MAGUX_VER Dropdown)
os_type VARCHAR(100) NO - '' 서버 설치 운영체제 (OS Dropdown)
installed_at DATE YES MUL (Index) NULL 실 납품/설치된 일자
description TEXT YES - NULL 설치 제품 세부 비고 사항
created_by INT UNSIGNED NO MUL (FK) None 등록 임직원 ID (REFERENCES member(id))
created_at DATETIME NO - CURRENT_TIMESTAMP 정보 등록 시각
updated_at DATETIME NO - CURRENT_TIMESTAMP 정보 수정 시각

대항목 카테고리 테이블 (mng_category)

Table Schema: mng_category
필드명 데이터 타입 Null Key 기본값 설명 / 비고
id INT UNSIGNED NO PRI (Auto Inc) NULL 대항목 고유 ID (PK)
code VARCHAR(100) NO UNI None 대항목 매핑 코드 (예: HW, OS, MAGUX_VER)
name VARCHAR(255) NO - None 대항목 라벨명 (예: 마구스 버전, 설치 OS)
description TEXT YES - NULL 대항목 설명 비고
created_at DATETIME NO - CURRENT_TIMESTAMP 정보 등록 시각
updated_at DATETIME NO - CURRENT_TIMESTAMP 정보 수정 시각

세부 항목 테이블 (mng_item)

Table Schema: mng_item
필드명 데이터 타입 Null Key 기본값 설명 / 비고
id INT UNSIGNED NO PRI (Auto Inc) NULL 세부 항목 고유 ID (PK)
category_id INT UNSIGNED NO MUL (FK) None 상위 대항목 ID (REFERENCES mng_category(id) ON DELETE CASCADE)
code VARCHAR(100) NO UNI None 세부 항목 고유 코드 (예: M1, ROCKY9, V1.0)
name VARCHAR(255) NO - None 세부 항목 명칭 (예: Rocky Linux 9.2)
is_use ENUM('Y','N') NO - 'Y' 사용 가능 유무 플래그
description TEXT YES - NULL 세부 항목에 대한 상세 정보 설명
created_at DATETIME NO - CURRENT_TIMESTAMP 정보 등록 시각
updated_at DATETIME NO - CURRENT_TIMESTAMP 정보 수정 시각

제품 업그레이드 이력 테이블 (product_history)

Table Schema: product_history
필드명 데이터 타입 Null Key 기본값 설명 / 비고
id INT UNSIGNED NO PRI (Auto Inc) NULL 이력 고유 번호 (PK)
product_id INT UNSIGNED NO MUL (FK) None 제품 ID (REFERENCES product(id) ON DELETE CASCADE)
old_version_code VARCHAR(100) YES - NULL 이전 제품 버전 코드
old_version_name VARCHAR(255) YES - NULL 이전 제품 버전 이름
new_version_code VARCHAR(100) YES - NULL 변경 새 버전 코드
new_version_name VARCHAR(255) YES - NULL 변경 새 버전 이름
old_os_code VARCHAR(100) YES - NULL 이전 설치 OS 코드
old_os_name VARCHAR(255) YES - NULL 이전 설치 OS 이름
new_os_code VARCHAR(100) YES - NULL 변경 새 OS 코드
new_os_name VARCHAR(255) YES - NULL 변경 새 OS 이름
notes TEXT YES - NULL 업그레이드 관련 참고사항/메모
created_by INT UNSIGNED NO MUL (FK) None 작업 실행 임직원 ID (REFERENCES member(id))
created_at DATETIME NO - CURRENT_TIMESTAMP 이력 기록 등록 시각
